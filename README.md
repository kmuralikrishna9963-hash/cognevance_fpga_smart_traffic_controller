# FPGA-Based Smart Traffic Controller

A Smart Traffic Controller implemented on FPGA using **Verilog HDL** and **Intel Quartus Prime Lite**. The system uses a **Finite State Machine (FSM)** to control traffic signals based on vehicle detection from the side road. The project includes simulation, synthesis, timing analysis, and complete project documentation.

---

# Project Objectives

The main objectives of this project are:

- Design a Smart Traffic Controller using Verilog HDL.
- Implement the controller using a Finite State Machine (FSM).
- Control traffic lights for the main road and side road efficiently.
- Detect vehicles on the side road using a sensor input.
- Simulate the design to verify correct functionality.
- Perform synthesis and timing analysis using Intel Quartus Prime Lite.
- Demonstrate FPGA-based digital system implementation with optimized hardware resources.

---

# Project Workflow

The project was completed following the standard FPGA design flow:

1. Requirement Analysis
   - Define the traffic control requirements and signal sequence.

2. FSM Design
   - Design the Finite State Machine representing different traffic light states.

3. Verilog HDL Coding
   - Develop the traffic controller using Verilog HDL.

4. Testbench Development
   - Create a testbench to verify system functionality under different conditions.

5. Functional Simulation
   - Simulate the design and verify state transitions and output signals.

6. Synthesis
   - Compile the design using Intel Quartus Prime Lite.

7. Timing Analysis
   - Analyze setup time, hold time, and maximum operating frequency using TimeQuest Timing Analyzer.

8. Documentation
   - Prepare project documentation, diagrams, and reports.

9. GitHub Repository
   - Organize all source code, reports, documentation, and screenshots into a professional repository.
